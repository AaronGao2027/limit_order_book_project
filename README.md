# Spy Limit Order Book Reconstruction
The historical MBO data used in this analysis covers the entire trading day of the SPY ETF on March 23, 2026 (2026-03-23). It is stored in a parquet file that is too big to upload to Github (2.31 GB). I have uploaded the parquet file to a google drive and attached a link that can be used to access the shared google drive and download the parquet file. Using the parquet file, all the results generated in this code are reproducible on a local machine.
